# 2. Requirement Analysis

This folder contains the requirement analysis documents for the **Smart Lender – Loan Eligibility Prediction System**.

The requirement analysis phase identifies the needs, expectations, and technical requirements required for developing an intelligent machine learning-based loan prediction application.

## Documents Included

- Requirement Gathering
- Functional Requirements
- Non-Functional Requirements
- Hardware and Software Requirements

These documents define the system objectives, user expectations, features, and resources required for successful project implementation.
